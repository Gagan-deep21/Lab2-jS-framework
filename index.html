JavaScript Framework Lab 2

This project demonstrates function creation with and without return types, higher-order functions, and the ES6 module system.

To run the project, follow the instructions below:

    1. Clone the repository:
       git clone https://github.com/Gagan-deep21/Lab2-jS-framework.git
       cd Lab2-jS-framework

    2. Install dependencies:
       npm install (if applicable)

    3. Run the project:
       node main.js
